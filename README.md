# Web-Development-Technology-Practicals

🌐 Web Development Technology Lab
📘 Overview

This repository contains all practical assignments for the Web Development Technology Lab under the Department of Computer Science Engineering & Cyber Security. The lab focuses on building fundamental to advanced web development skills using HTML, CSS, Bootstrap, Tailwind CSS, JavaScript, and GitHub.

🎯 Objectives
To understand and implement basic web design concepts
To develop static and dynamic web pages
To learn responsive web design techniques
To gain hands-on experience with modern frameworks like Bootstrap and Tailwind CSS
To implement client-side validation and interactive web applications


🧪 List of Experiments

🔹 HTML & CSS (CO1 & CO2)
Practical#1.Create a timetable using HTML table tags
Practical#2.Design student and employee registration forms
Practical#3.Design web pages using <div> and CSS properties
Practical#4.Create a static college webpage (Vision, Mission, Courses, etc.)
Practical#5.Build a responsive personal portfolio using media queries

🔹 Bootstrap (CO3)
Practical#6.Design a web page using HTML and Bootstrap
Practical#7.Create registration and sign-in forms using Bootstrap
Practical#8.Develop a responsive college webpage using Bootstrap
Practical#9.Design a checkout form
Practical#10.Create an interactive product listing card

🔹 Tailwind CSS & Deployment (CO4)
Practical#11.Design a navigation bar with dropdown using Tailwind CSS

🔹 JavaScript (CO5)
